# 섹터-레짐 매핑 실증 검증 보고서

> 작성일: 2026-02-25
> 데이터 범위: 2023-02-27 ~ 2026-02-24 (약 3년)
> 분석 유효 월수: 31개월 (Indeterminate 7개월 제외)

---

## 1. 배경 및 목적

`config/sector_map.yml`의 국면별 섹터 배정은 경제 이론 직관으로 수동 입력된 값으로,
한국 시장 데이터 기반 실증 검증이 전혀 없었다.

본 검증은 실제 KRX 섹터 인덱스 수익률과 레짐 이력을 대조하여 배정의 유효성을 측정하고,
근거 없는 배정을 수정하는 것을 목표로 한다.

---

## 2. 분석 방법론

### 레짐 분류
- **성장 지표**: KOSIS 경기선행지수순환변동치 (`DT_1C8015`, 3개월 이동평균 방향)
- **물가 지표**: KOSIS CPI YoY (`DT_1J22003`, 3개월 이동평균 방향)
- **레짐 매트릭스**:

| 성장 방향 | 물가 방향 | 레짐 |
|---|---|---|
| Up | Down | Recovery |
| Up | Up | Expansion |
| Down | Up | Slowdown |
| Down | Down | Contraction |

잠정치(`is_provisional=True`)는 분석에서 제외.

### 성과 측정
- 각 섹터의 **월수익률** 계산 (월말 종가 기준)
- **초과수익** = 섹터 수익률 − KOSPI 수익률
- 레짐별 평균 초과수익 집계 → 섹터 순위 산출
- 분석 도구: [`scripts/validate_sector_mapping.py`](../scripts/validate_sector_mapping.py)
